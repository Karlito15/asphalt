# NAS 

## Composer  

### Install
``` shell
php84 /volume3/web/WWW-CDN/composer.phar install --no-dev --no-scripts --optimize-autoloader
```

### Update
``` shell
php84 /volume3/web/WWW-CDN/composer.phar update --no-dev --no-scripts --optimize-autoloader
```

### Dump Local
``` shell
php84 /volume3/web/WWW-CDN/composer.phar dump-env prod
```
