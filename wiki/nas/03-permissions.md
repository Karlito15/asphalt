# NAS 

## Permissions  

### Create Directory
``` shell
mkdir vars
```

### Modify Permissions
``` shell
sudo chmod -R a+rw documents/
```
OR
``` shell
sudo chmod -R 0777 documents/
```

``` shell
sudo chmod -R a+rw vars/
```
OR
``` shell
sudo chmod -R 0777 vars/
```

``` shell
sudo chmod -R a+rw app/public/
```
OR
``` shell
sudo chmod -R 0777 app/public/
```
